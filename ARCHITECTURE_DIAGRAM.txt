┌──────────────────────────────────────────────────────────────────────────────┐
│                         AGENT SEQUENCE                                       │
└──────────────────────────────────────────────────────────────────────────────┘

    Step 1: Reception Agent
    ┌─────────────────────────────────────┐
    │ • Collects client information       │
    │ • Health history                    │
    │ • Fitness background                │
    │ • Goals & preferences               │
    │                                     │
    │ Context: None (first in sequence)   │
    │ Output: Client Profile              │
    └──────────────┬──────────────────────┘
                   │
                   │ Passes: Client intake data
                   ▼
    Step 2: Body Scanner Agent
    ┌─────────────────────────────────────┐
    │ • Analyzes body posture             │
    │ • Movement patterns                 │
    │ • Mobility assessments              │
    │ • Kinesiology analysis              │
    │                                     │
    │ Context: Client intake              │
    │ Output: Body Analysis               │
    └──────────────┬──────────────────────┘
                   │
                   │ Passes: Intake + Body analysis
                   ▼
    Step 3: PT Agent
    ┌─────────────────────────────────────┐
    │ • Creates training plan             │
    │ • Exercise programming              │
    │ • Sets, reps, progression           │
    │ • Safety considerations             │
    │                                     │
    │ Context: Intake + Body analysis     │
    │ Output: Training Plan               │
    └──────────────┬──────────────────────┘
                   │
                   │ Passes: Intake + Body + Training
                   ▼
    Step 4: Nutrition Agent
    ┌─────────────────────────────────────┐
    │ • Creates nutrition plan            │
    │ • Meal planning                     │
    │ • Macronutrient targets             │
    │ • Meal timing                       │
    │                                     │
    │ Context: Intake + Body + Training   │
    │ Output: Nutrition Plan              │
    └──────────────┬──────────────────────┘
                   │
                   │ Passes: ALL previous information
                   ▼
    Step 5: Head Coach Agent (Coordinator)
    ┌─────────────────────────────────────┐
    │ • Coordinates all agents            │
    │ • Integrates all information        │
    │ • Creates final program             │
    │ • Ensures coherence                 │
    │                                     │
    │ Tools:                              │
    │   • Google Search                   │
    │   • PreloadMemoryTool               │
    │   • A2A Communication Tools:        │
    │     - get_client_information()      │
    │     - get_body_analysis()           │
    │     - get_training_plan()           │
    │     - get_nutrition_plan()          │
    │                                     │
    │ Context: EVERYTHING                 │
    │ Output: Final Integrated Program    │
    └──────────────┬──────────────────────┘
                   │
                   ▼
        ┌──────────────────────┐
        │   FINAL PROGRAM      │
        │                      │
        │ • Complete training  │
        │ • Complete nutrition │
        │ • Integrated plan    │
        └──────────────────────┘
